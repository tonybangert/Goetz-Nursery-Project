# Goetz's Nursery — Bayside Tree Planner

Interactive tree selector for Goetz's Nursery (Hartford, WI), curated for a Bayside, WI property along the Lake Michigan corridor (USDA Zone 5a / 4b).

Browse 38 deciduous and evergreen trees with full specs, pros/cons, fall color, sun/soil requirements, and Bayside-specific siting notes.

## Stack

- **React 18** with hooks
- **Vite 5** dev server / bundler
- Inline styles + Google Fonts (Cormorant Garamond + DM Sans) — no Tailwind, no UI lib
- Single component: `src/App.jsx`

## Develop

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build → dist/
npm run preview  # preview built output
```
